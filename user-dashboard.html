Peraturan User Dashboard

Informasi dan Ketentuan Penggunaan

Peraturan ini dibuat agar data tugas, report, leaderboard, dan histori kerja tetap konsisten di dalam sistem.

Jangan mengubah nama di tengah pengerjaan

User tidak disarankan mengubah informasi nama saat tugas, project, QC, atau report sedang berjalan. Perubahan nama di tengah proses dapat membuat data lama tidak sinkron dengan leaderboard, report, dan histori.

Perubahan profil berlaku di awal setelah login

Perubahan data profil sebaiknya dilakukan di awal penggunaan setelah login pertama, sebelum user mulai mengerjakan tugas. Dengan begitu sistem dapat mendeteksi identitas user secara lebih konsisten.

Data tugas yang sudah berjalan memakai identitas awal

Tugas, status project, history task, report, dan leaderboard dapat tetap memakai data identitas yang tercatat saat tugas dibuat atau dinilai.

Hubungi admin untuk perubahan penting

Jika ada kesalahan data profil penting seperti nama, divisi, atau posisi, user disarankan menghubungi admin agar perubahan dapat diperiksa dan disesuaikan dengan data sistem.
